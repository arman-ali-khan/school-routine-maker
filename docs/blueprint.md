# **App Name**: Routinely

## Core Features:

- Drag-and-Drop Schedule Builder: Interactive drag-and-drop interface to build and manage weekly class schedules.
- Customizable Time Slots: Customize time slots with adjustable start and end times, including break time designation.
- Subject Management: Create and manage subjects with details like subject name, teacher, subject code, and color coding for easy identification.
- Routine Cell Actions: Quick actions on each subject within the schedule, like edit, delete, copy, and move, along with detailed tooltips on hover.
- Local Data Management: Automatically save data locally and allow exporting/importing schedules in JSON format for backup and sharing.

## Style Guidelines:

- Primary color: Use a calming blue (#64B5F6) to provide a sense of organization and tranquility, reducing the stress of schedule management. This blue is versatile and inviting.
- Background color: A very light blue (#E3F2FD) to ensure a clean and non-distracting environment.
- Accent color: Use a muted green (#81C784), as this falls approximately 30 degrees to the 'left' of blue on the color wheel. The selected saturation creates a more subdued contrast than would be expected in a dark color scheme.
- Use clear and modern typography.
- Employ simple, intuitive icons for actions like edit, delete, and copy.
- Maintain a clean, grid-based layout for the weekly schedule.
- Use subtle animations and transitions when dragging and dropping subjects.
